# Smartinovate — Next.js 14 Starter

This single-file starter contains a ready-to-use Next.js App Router project scaffold (TailwindCSS) and the main files you need to reproduce the homepage you pasted. Copy the files into a new Next.js 14 app (or use `create-next-app`) and adjust assets, domain, and analytics keys.

---

## Project structure (what this starter includes)

```
smartinovate/
├─ package.json
├─ next.config.js
├─ postcss.config.js
├─ tailwind.config.js
├─ app/
```

## 13) Public assets

Place logos and svgs under `/public/icons/` and images under `/public/images/` and `/public/projects/`. Optimize them (SVGO for SVGs, and convert large raster to WebP).

## 14) Deployment

1. Push to GitHub.
2. Connect repository to Vercel (recommended for Next.js). Set environment variables if needed (analytics keys, API endpoints).
3. Configure domain on Vercel or via Cloudflare (point A / CNAME to Vercel). Enable HTTPS.

## 15) SEO & Analytics quick setup

- Add a `script` for Google Analytics via `next/script` with `strategy="afterInteractive"` and use your GA id.
- Add `app/sitemap.xml` generation (or use `next-sitemap` package) and upload to Google Search Console.
- Add JSON-LD (Organization) into `app/layout.tsx` with `next/script` strategy afterInteractive.
